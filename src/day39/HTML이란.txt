[주요마크업]
    - 마크업 형태 종류
        1. <마크업명> 내용물 </마크업명>
        2. <마크업명 />
    - 마크업의 속성이란
        1. 형태 : <마크업명 속성명=값 속성명=값 >
        2. 목적 : 마크업의 고유한 기능을 적용할수 있다.

1.<html> </html>        : html시작과 html끝 알리는 마크업
2.<head> </head>        : html문서의 다양한 옵션정보 읽어드리는 마크업
3.<title> </title>      : html문서의 제목
4.<body> </body>        : html문서의 *본문*

5.<h1> </h1>            : h1 ~ h6 까지 제목 마크업 , 글자크기/굵게 제공
6.<p> </p>              : 문단 마크업 , 구분/여백 제공
7.<br />                : 줄바꿈 마크업
8.<hr />                : 가로 구분선 마크업
9.<ul> </ul>            : unorder list 순서없는목록
10.<ol> </ol>           : order list 순서있는목록
11.<li> </li>           : 목록의 항목 , *ul 또는 ol 마크업 안에서 사용된다.

12.<img src="" />       : 이미지 마크업
    - 속성
        src=" "         : 표시할 이미지의 파일 경로 , 확장자명 포함
    - 경로기준 종류
        1. 절대경로 : C:\Users\admin\Desktop\kim\src\day39\짱구.jpg
        2. 상대경로 : (현재파일기준)    ../ : 상위로 이동 뜻    ./ : 현재경로 뜻(생략가능)
            - 같은 폴더내 존재하면 : ./파일명.확장자
            - 상위 폴더내 존재하면 : ../파일명.확장자
            - 하위 폴더내 존재하면 : ./하위폴더명/파일명.확장자
            - 형제 폴더내 존재하면 : ../형제폴더명/파일명.확장자

13. <audio> </audio> , <video> </video>
    - 속성 :
        - controls      : 오디오/비디오 실행 컨트롤러 표시 여부
        - autoplay      : 오디오/비디오 자동 실행 , *브라우저에 정책에 따라 사용이 불가능.
        - muted         : 음소거 와 자동실행은 같이 사용 가능.
